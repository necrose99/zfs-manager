            <header>
                <h2>Description</h2>
            </header>
            <p id="description" itemprop="description">ZFS Manager is a menu driven utility for managing ZFS Filesystems written in python. Features include:  <br>
<br>
Creating/Deleting/Managing ZPOOLs<br>
Creating/Deleting/Managing Datasets<br>
Creating/Deleting/Managing Snapshots<br>
Creating/Removing ZILs/Cache Drives<br>
Managing ZFS Options (compression, NFS)<br>
Creating/Deleting/Managing ZFS Filesystems<br>
<br>
Future Features:<br>
<br>
ZFS Replication<br>
<br>
Requirements: Python 2.x, ZFS Filesystem<br>
<br>
Obligatory Warning: I am not responsible for lost/corrupted data due to the usage of this product.</p>
